# note-pipeline

AI（Claude Code）を活用した note.com 記事自動生成パイプライン。

## プロジェクト概要

- noteで「AI自動化×大学生」をテーマに記事を発信
- GitHub Actionsで記事生成を自動化
- 人間レビュー→手動投稿のハイブリッド運用

## 記事タイプ

1. **実録記事（jituroku）**: 案件受注→自動化→納品の全記録。週1本。有料1,500〜2,000円
2. **速報記事（sokuho）**: AI新機能の即日レビュー。不定期。有料500〜800円
3. **ノウハウ記事（knowhow）**: Claude Code活用ガイド等。月2本。無料 or 前半無料+後半有料

## ルール

- 記事の文体: 大学生が自分の言葉で書いているトーン。AI感を出さない
- 「〜と言えるでしょう」「〜ではないでしょうか」等のAI定型フレーズ禁止
- 数字・実績は正確に。盛らない
- クライアント情報は必ず匿名化
- noteの装飾: 見出し（##, ###）、太字（**）、リスト（-）のみ使用。表・コードブロックはnoteで崩れるため避ける
- コードを載せる場合はインデント付きテキストで代用

## スキル

- `/note-run` : 全自動パイプライン。トピック選定→競合調査→記事生成→編集→最終稿出力を1コマンドで実行

## フォルダ構成

- `context/` : プロフィール・既存記事情報・文体サンプル・収益化戦略・パフォーマンスデータ
- `drafts/` : 生成された草稿（jituroku/sokuho/knowhow）
- `published/` : 投稿済み記事ログ
- `.github/prompts/` : 記事タイプ別のプロンプト
- `.github/workflows/` : GitHub Actionsワークフロー
