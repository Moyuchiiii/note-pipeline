# note-pipeline

Claude Code で note.com の記事を全自動生成するパイプライン。

トピック選定 → 競合調査 → 記事執筆 → 品質チェック → 最終稿出力まで、`/note-run` コマンド1つで完結する。

## できること

- AI最新ニュースを自動スキャンして、記事にすべきトピックを選定
- 5項目スコアリング + PDCAフィードバックで有料/無料を自動判定
- noteエディタ対応のHTML形式で出力（見出し・引用・コード・リスト等の装飾付き）
- 100点ルーブリックによる品質ゲート（60点未満は自動rewrite）
- タイトルA/Bテスト（3本生成 → スコアリングで最適選定）
- 投稿タイミングの最適化提案
- Notion連携（記事管理DB + ネタ帳DB）で進捗・パフォーマンスを一元管理
- 過去記事の成功パターンを自動学習してPDCAサイクルを回す

## セットアップ

### 前提条件

- [Claude Code](https://code.claude.com/) がインストール済み
- Claude Pro / Max プランに加入済み
- （任意）Notion MCP サーバーが設定済み

### インストール

```bash
git clone https://github.com/Moyuchiiii/note-pipeline.git
cd note-pipeline
claude
```

### 使い方

```bash
/note-run
```

これだけ。あとは全自動で以下が実行される：

1. **Phase 0** — コンテキスト読み込み + PDCA自動学習
2. **Phase 1** — AIニューストレンドスキャン
3. **Phase 2** — トピック選定（スコアリング + バランス制御）
4. **Phase 3** — 競合調査
5. **Phase 4** — 構成設計 + タイトルA/Bテスト
6. **Phase 5** — 執筆（HTML形式 + 装飾フル活用）
7. **Phase 6** — 品質ゲート（7スウィープ + 100点ルーブリック）
8. **Phase 7** — SEO最適化
9. **Phase 8** — 保存 + 投稿ガイド出力
10. **Phase 9** — PDCAフィードバック記録

## 出力されるもの

`/note-run` 実行後、以下が出力される：

| 出力 | 内容 |
|------|------|
| `.md` ファイル | メタ情報付きMarkdown（バックアップ用） |
| `.html` ファイル | noteエディタにペーストするHTML |
| タグ一覧 | noteのハッシュタグ用 |
| 有料ライン位置 | 有料エリアの設定位置 |
| サムネプロンプト | Nano Banana用の画像生成プロンプト |
| 投稿タイミング | 最適な曜日・時間帯の提案 |
| 品質スコア | 100点満点のルーブリック採点結果 |

## 投稿フロー（人間がやること）

1. サムネイルを生成（出力されたプロンプトを使用）
2. noteエディタを開いてタイトル入力
3. HTMLをエディタにペースト（装飾自動反映）
4. サムネイル画像をアップロード
5. 公開設定（タグ・有料設定・有料ライン）
6. 投稿ボタンを押す

## PDCA自動学習

`/note-run` は実行するたびに過去記事のパフォーマンスを自動分析する。

- スキ数が多かった記事のテーマ・タイトルパターンを学習
- 成功パターンに類似するトピックにボーナス加算（+2点）
- 反応が悪かったテーマにはペナルティ（-2点）
- 文体サンプル（voice-samples.md）もスキ数上位記事から自動更新
- 無料/有料のバランスを自動制御（連投防止・初期フォロワー制限）

## Notion連携

2つのDBで記事のライフサイクルを管理：

- **note記事管理DB** — 記事のステータス・価格・PV・スキ数・売上・PDCAメモ
- **ネタ帳DB** — スキャンで見つけたネタ候補のストック（未使用/採用済み/見送り）

## 品質ゲート

100点ルーブリックで記事を自動採点：

| カテゴリ | 配点 |
|----------|------|
| コンテンツ品質 | 30点 |
| SEO | 25点 |
| 読者体験 | 20点 |
| voice一貫性 | 15点 |
| リスク | 10点 |

60点未満は自動rewrite（最大2回）。2回でもダメならユーザーに報告。

## 記事タイプ

| タイプ | 内容 | 目安文字数 | 価格帯 |
|--------|------|-----------|--------|
| 速報（sokuho） | AI新機能の即日レビュー | 800〜1,500字 | 500円 |
| 実録（jituroku） | 案件受注→自動化→納品の全記録 | 2,000〜4,000字 | 1,500〜2,000円 |
| ノウハウ（knowhow） | Claude Code活用ガイド等 | 2,500〜5,000字 | 0〜2,000円 |
