結論から

3月24日、Anthropicが Computer Use をリリースした。

要は、ClaudeがPCの画面を見て、マウスとキーボードを操作して、タスクを勝手にやってくれる機能。

先週出たDispatchと組み合わせると、スマホからタスクを投げて、PCで自動実行して、終わったら結果だけ受け取る——みたいなことができる。

副業やってる身からすると、これは結構デカい。

何ができるのか

公式が出してる例はこんな感じ。

通勤中にスマホから「朝のブリーフィング作って」と指示 → PCがメール・カレンダー・ニュースをまとめてくれる
IDEでコード修正 → テスト実行 → PR作成まで自動
ブラウザを開いてスプレッドシートにデータ入力
週次メトリクスの収集と整理

ポイントは、SlackやGoogleカレンダーみたいにAPI連携できるアプリはそっちを優先して、「直接つなげないアプリ」のときだけ画面操作に切り替えるところ。全部ゴリゴリ画面操作するわけじゃないから、思ったより賢い。

使うための条件

macOS限定（Windowsは今後対応予定）
Claude Pro または Max プラン
設定 → Desktop app → General → Computer Use をオン
macOSの「アクセシビリティ」と「画面収録」の権限を許可
デスクトップアプリを起動した状態にしておく（スリープNG）

セキュリティ面では、アプリごとに「このアプリ使っていい？」って聞いてくる。勝手に全部触られるわけじゃないから、そこは安心。

副業でどう使えそうか

自分が真っ先にやりたいのは3つ。

1. クラウドワークスのメッセージ確認

今は/loopで新着案件チェックを自動化してるけど、受注後のクライアントとのやり取りは手動だった。Computer Useなら「ブラウザでクラウドワークスを開いて → 受注中の案件メッセージを確認 → 返信が必要なものをリストアップ」まで自動でやれる。

2. Excelの報告書作成

クライアントから「Excelで出して」って言われることが割とある。今まではClaude Codeでデータを作って → 自分でExcelに貼り付けてた。Computer Useなら「Excelを開いて → データ入力 → グラフ作成 → 保存」まで全自動。

Before: データ作成30分 + Excel整形20分 = 50分
After: 指示を1回送るだけ。待ち時間のみ

3. APIがないサイトのリサーチ

競合調査とか、特定のサイトの価格チェックとか。APIが公開されてないサイトは今まで手動でやるしかなかったけど、Computer Useなら「ブラウザで開いて → 情報を拾って → まとめて」ができる。

正直な感想

まだリサーチプレビューだし、macOSだけ。Anthropic自身が「コーディングやテキスト処理に比べると、画面操作はまだ初期段階」と言ってる。

いきなり重要な作業を任せるのは怖いから、失敗しても困らないタスクから試すのが良さそう。個人情報が入ってるアプリは慎重にいったほうがいい。

ただ、方向性として「AIがPCを操作する」は確実に来てる。今のうちに触っておいて損はない。

まとめ

Claude Computer Useは「AIに画面操作ごと任せる」新機能。

Dispatch × Computer Use で、「スマホからタスクを投げて → PCで自動実行 → 結果だけ受け取る」が現実になった。副業の「手を動かさない時間」がまた減る。

次回は、実際にComputer Useで副業タスクを自動化した体験記を書く予定。フォローしておくと通知が届きます。
